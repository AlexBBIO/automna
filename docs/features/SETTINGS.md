# Settings Panel

> Last updated: 2026-02-11

## Current Implementation

**Component:** `components/SettingsPanel.tsx`

Displays agent info fetched from `GET /api/user/gateway`:
- Agent email address (with copy button)
- Phone number (if provisioned)
- Browserbase context ID
- App name and region
