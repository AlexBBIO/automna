- Short, direct, no fluff
- Specific product details: fully agentic, built-in email, browsing, phone calls coming soon, built on OpenClaw, dedicated server

### Key Lessons
- Don't offer "logins", offer "codes" (more exclusive)
- Don't presume they'll make content. "If you dig it, could be a fun X" not "I think this would make a great video"
- Be specific about what the product does. Vague = forgettable.
- "Took a lot of work" not "took months" or any specific timeframe
- Don't say "no setup, no servers" when it literally runs on a server
- End every message on a clear ask
